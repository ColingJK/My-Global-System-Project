# My-Global-System-Project
Automation scripts for Raspberry Pi server management and logic systems in Python and Luau.
# DevSecOps & Infrastructure Lab

##  Projects Overview
This repository contains the core logic for my Raspberry Pi 3B server deployment and automation experiments.

###  Security & Ops (Python)
- **backup.py**: Automated directory mirroring for disaster recovery.
- **monitor.py**: Resource telemetry for CPU/RAM health checks.
- **Firewall Config**: Hardened Linux environment using UFW.

###  Logic Systems (Luau/Roblox)
- **Logic_Systems.lua**: Demonstrates event-driven programming and state management.

##  Hardware
- **Platform:** Raspberry Pi 3B (Element 14)
- **Specs:** 1GB RAM / ARMv8
- **Troubleshooting:** Successfully diagnosed voltage-drop issues in power delivery.
